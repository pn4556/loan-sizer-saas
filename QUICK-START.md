## 🎯 Run It Now (3 Steps)

### Step 1: Install Python Dependencies
```bash
cd ~/workspace/loan-sizer-automation
pip install -r backend/requirements.txt
```

### Step 2: Launch the System
```bash
./start.sh
```

This will:
- Start the backend API on http://localhost:5050
- Open the dashboard in your browser
- You're ready to demo!

### Step 3: Try a Demo
1. Click "✅ Approve Scenario" button
2. Click "🚀 Process Application"
3. Watch the magic happen in ~2 seconds

## 🔌 API Endpoints

The backend provides:
- `POST /api/extract` - Extract data from email
- `POST /api/process` - Full pipeline (extract → evaluate → email)
- `GET /api/demo/scenarios` - Get test scenarios

All documented in README.md
